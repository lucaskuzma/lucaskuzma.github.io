Research

  • Master's thesis at UPF

    Designed and implemented user interfaces for a real-time concatenative synthesis performance and composition system. Combined Flash with OSC to facilitate user interaction with a native audio mosaicking engine.

  • Master's thesis at UCLA

    Created a quasi-neural network of autonomous, sound-making devices interacting with each other and with the sounds in their environment. Programmed all devices in C, using Atmel AVR microprocessors. Designed all circuitry. Used the Processing environment to create simulations of the network.
